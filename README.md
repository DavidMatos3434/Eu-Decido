# EU DECIDO - Sistema de Governação Digital

Este é um projeto Monorepo que contém a aplicação móvel (KMP) e a infraestrutura de backend (Supabase).

## Estrutura do Projeto

- **/composeApp**: Interface de utilizador Android/iOS (Jetpack Compose).
- **/shared**: Lógica de negócio e base de dados local (SQLDelight).
- **/backend**: Infraestrutura e lógica de servidor (Supabase).
- **/docs**: Documentação técnica e funcional.
